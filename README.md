#### ClickBoot App (root only)
#
#
#### FEATURES
  - Normal Reboot
  - Soft reboot Simply restart the android UI by Killing zygote process
  - Safe Mode automatic write buil.prop "setprop persist.sys.safemode 1" !!Not Recomended!!
  - Recovery mode
  - Fastboot mode
  - Power Off
